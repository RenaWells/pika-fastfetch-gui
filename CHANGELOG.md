# Changelog

All notable changes to Pika Fastfetch GUI are documented here.

This project follows the package versions used by the PikaOS apt repository. Debian packaging metadata is also kept in `debian/changelog`.

## 1.0.6+pika1 - 2026-06-02

- Added a native libadwaita About dialog.
- Added project website, issue link, GPL license, credits, and debug details to the About dialog.

## 1.0.5+pika1 - 2026-06-02

- Added the recommended package-manager install instructions for PikaOS users.
- Updated package maintainer metadata.

## 1.0.4+pika1 - 2026-06-02

- Used an absolute PNG icon path in the desktop entry for better launcher compatibility.

## 1.0.3+pika1 - 2026-06-02

- Added PNG hicolor app icons for application menu compatibility.

## 1.0.2+pika1 - 2026-06-02

- Refreshed the app icon with an original PikaOS-inspired mark.

## 1.0.1+pika1 - 2026-06-02

- Added a dedicated hicolor app icon for Linux app menus.

## 1.0.0+pika1 - 2026-06-02

- Initial PikaOS package.
- Added the customizable GTK 4/libadwaita fastfetch-style system overview.
